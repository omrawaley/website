# Personal Website

This is my personal website, where you can learn everything about me. It features some of my projects as well as a Markdown blog that I update with new findings and reflections.

<img width="800" height="" alt="website" src="https://github.com/user-attachments/assets/af98f649-9788-499d-aae5-a28437a6db5a" />

I created this website because I needed a singular place to store my thoughts that other people could also access and gain insight from. I am to write things like guides, talk about different programming topics, and even post a few write-ups of different projects.

I built this website using NextJS, React, and Tailwind. It's currently deployed on Vercel.

## Running Locally

If you want to run this locally for whatever reason, follow these steps:

1. Clone the repository

```
git clone https://github.com/omrawaley/website
```

2. Install dependencies

```
pnpm i
```

3. Run development version

```
pnpm run dev
```
